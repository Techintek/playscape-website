Known Issues & Workarounds

We're constantly working to improve Playscape. Here are some known issues we're currently addressing:

Date Synchronization Issue Fix in Progress

In the Create Game view, the date selector does not automatically sync to today's date.

Workaround: Manually select today's date when creating a new game.

Expected Fix: In our next update (v1.2.1), scheduled for release on May 5, 2025.

Delete Account Functionality Under Investigation

The Delete Account option in User Settings is currently not functioning as expected.

Workaround: To delete your account, please contact our support team at support@playscape-app.com with the subject line "Account Deletion Request".

Expected Fix: We're actively investigating this issue and expect to have it resolved in version 1.3.0.

Logout Redirect Fix in Progress

When logging out, the app successfully logs you out but does not properly redirect to the login/splash screen.

Workaround: After logging out, manually close and reopen the app to return to the login screen.

Expected Fix: This will be addressed in our next update (v1.2.1), coming May 5, 2025.

Message Notification Count Fixed in v1.2.0

Message notification counts sometimes showed incorrect numbers of unread messages.

Status: This issue has been resolved in the latest app version. Please ensure you are running version 1.2.0 or higher.

Update Schedule

We release updates regularly to improve performance, fix bugs, and introduce new features:

  • Version 1.2.1 - Expected May 5, 2025: Fix for date synchronization and logout redirect issues
  • Version 1.3.0 - Expected June, 2025: Major update including account management improvements and new community features

To ensure you have the latest improvements, please keep your app updated through the App Store.

Frequently Asked Questions

How do I create a new game?

To create a new game:

  1. Go to the "My Hub" tab
  2. Tap "Create or join a game to get started"
  3. Fill in the game details including title, description, sport type, location, date, time, and skill level
  4. Set the maximum number of participants
  5. Choose whether the game should be public or invite-only
  6. Tap "Create Game"
How do I join an existing game?

You can join existing games in a few ways:

  • From My Hub: Browse the list of games and tap on one you'd like to join, then tap "Join"
  • Through Search: Use the search function to find games by sport, location, or date
  • Via Invites: If someone has invited you to a game, you can accept the invitation from the Network tab
How do I message other players?

To message another player:

  1. Go to the "Network" tab
  2. Tap on the person you want to message (or tap the "+" icon to start a new conversation)
  3. Type your message and tap send

You can also message players from within a game by tapping on their profile and selecting "Message".

Can I change my skill level?

Yes, you can update your skill level for each sport separately:

  1. Go to your profile by tapping your profile icon
  2. Tap "Edit Profile"
  3. Scroll to "My Sports" section
  4. Select the sport and adjust your skill level
  5. Save your changes
Is there an Android version available?

Currently, Playscape is only available for iOS devices. We're actively developing an Android version, which we expect to release in late 2025. Sign up for our newsletter on the homepage to be notified when the Android version becomes available.
